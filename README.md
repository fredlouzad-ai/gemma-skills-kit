# Gemma 4 Agent Skills - Starter Kit v1.0

Welcome to the Starter Kit for building local, privacy-first agentic workflows using Google's Gemma 4 and the **Google AI Edge Gallery**.

## Included Skills

### 1. local-file-parser
Allows Gemma to parse structured CSVs and JSONs locally. Perfect for privacy-focused data analysis (e.g., HR reps, finance).

### 2. markdown-report-generator
Takes Gemma's analysis and formats it into a clean Markdown structure. Excellent for offline inspectors or contractors needing to generate standard reports on-site.

### 3. local-network-fetcher
Enables Gemma to query smart devices on the local Wi-Fi safely, rejecting outward internet requests to maintain zero-cloud privacy.

## How to Install in Google AI Edge Gallery
1. Open the **Google AI Edge Gallery** app.
2. Navigate to the **Agent Skills** tile.
3. Tap the (+) button and select the **Load skill from URL** option.
4. Paste the URL to the skill folder. 

**Example URLs to paste:**
- `https://fredlouzad-ai.github.io/gemma-skills-kit/local-file-parser`
- `https://fredlouzad-ai.github.io/gemma-skills-kit/local-network-fetcher`
- `https://fredlouzad-ai.github.io/gemma-skills-kit/markdown-report-generator`
